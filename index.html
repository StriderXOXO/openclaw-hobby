ARCHITECTURE

Architecture

Four-layer pipeline — fully automated from collection to delivery

Collection — Daemons

Three-source parallel monitoring — Podcast / YouTube / Twitter, 24/7

Storage

SQLite (zero-config default) or Feishu Bitable — pluggable backend

Analysis — Triage

LLM deep analysis — summaries, highlights, curated excerpts & topic tags

Decision — Heartbeat

AI Agent global scoring — picks the right moment, pushes to Feishu / Telegram
